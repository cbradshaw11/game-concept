# MVP Acceptance Checklist
## Slice 1: Sanctuary to Ring 1 Extraction

Date: 2026-03-09
Status: Pending

## Functional
1. Player can start in Sanctuary and open prep menu.
2. Player can equip one weapon and enter Ring 1.
3. Ring 1 spawns one valid encounter from seeded template.
4. Player can defeat encounter and collect rewards.
5. Player can extract and bank rewards in Sanctuary.

## Combat
1. Attack consumes stamina and deals expected damage.
2. Dodge grants invulnerability for configured window.
3. Guard reduces incoming damage by weapon guard efficiency.
4. Poise break triggers stagger state correctly.

## Progression
1. XP and loot use ring multipliers correctly.
2. Death in ring loses unbanked rewards.
3. Extraction preserves banked rewards.

## Stability
1. Save and reload preserves progression state.
2. Seed replay reproduces same encounter composition.
3. No blocker bugs in one full run loop.
